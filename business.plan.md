# Business Plan & Web Architecture: RT Software Factory

## Contexto del Proyecto

Soy Desarrollador Frontend (+5 años exp) y estoy lanzando **RT Software Factory**, una agencia de servicios web y automatización con IA basada en Santa Rosa, La Pampa. El modelo de negocio se basa en **SaaS propio** y **Servicios de Suscripción (Abonos Mensuales)**.

## Identidad de Marca

- **Nombre:** RT Software Factory (Desarrollo de Software & Automatización).
- **Eslogan:** "Hacemos que tu negocio trabaje por vos".
- **Estética:** Dark mode, profesional, cian neón, minimalista.

## Unidades de Negocio (SaaS)

### 1. StockAutos.com.ar

- **Target:** Concesionarias de autos.
- **Features:** Control de stock, multipublicación automática en redes sociales, generación de videos comerciales (Reels) y audios con IA para cada vehículo.
- **Valor:** Velocidad de venta y presencia profesional constante.

### 2. Alkila.com.ar

- **Target:** Inmobiliarias y Dueños directos.
- **Features:** Gestión de alquileres, bot de WhatsApp para filtrado de consultas y requisitos, administración de contratos.
- **Valor:** Reducción de carga operativa y respuesta 24/7.

### 3. RAM Informática (E-commerce Engine)

- **Target:** Retail/Comercios locales.
- **Features:** Tienda online con enriquecimiento de datos vía IA (descripciones y SEO automático), integración con Supabase y n8n.

## Modelo de Monetización

- **Setup Fee:** Pago inicial por configuración y carga de datos.
- **Suscripción Mensual:** Abonos escalables que incluyen hosting, soporte, actualizaciones de la IA y mantenimiento.

## Requerimientos para la Web (FluxIA.com.ar)

1. **Sección Hero:** Propuesta de valor clara (Automatización + IA).
2. **Sección Productos:** Cards para StockAutos, Alkila y RAM Informática con sus respectivos dominios.
3. **Sección Servicios IA:** Explicar integraciones con n8n, automatización de flujos de trabajo y bots de WhatsApp.
4. **Sección Beneficio Local:** Enfatizar el soporte en Santa Rosa y cercanía.
5. **Formulario de Contacto:** Lead magnet para pedir una "Demo Gratuita".

Unos tips finales antes de dárselo:
Variables de Entorno: Como ya tenés experiencia con n8n y Supabase, pedile que deje preparados los formularios para disparar webhooks de n8n.

Diferencial de "Code": Pedile específicamente que use Lucide-react para los íconos y Framer Motion para que la web de FluxIA tenga "fluidez" (haciendo honor al nombre).

## Instrucciones para Claude Code

1. Generar la estructura de archivos para un proyecto de Next.js + Tailwind CSS.
2. Crear componentes de UI modernos siguiendo la estética "FluxIA" (Glassmorphism, Dark mode).
3. Redactar el copy de venta para cada sección basándose en los datos anteriores.
4. Implementar una sección de precios con 3 niveles de abono (Básico, Pro, Enterprise).
